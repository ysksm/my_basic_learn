# CSSサンプル集

このフォルダには、CSSの基本的なプロパティとレイアウト手法を学ぶためのサンプルファイルが含まれています。
各ファイルは1つの概念に焦点を当てています。

## サンプル一覧

1. **01_selector.css** - セレクタの基本
   - 要素セレクタ
   - クラスセレクタ
   - IDセレクタ

2. **02_color.css** - 色の指定方法
   - カラーネーム
   - HEX（16進数）
   - RGB/RGBA

3. **03_box_model.css** - ボックスモデル
   - padding（内側の余白）
   - border（境界線）
   - margin（外側の余白）

4. **04_display.css** - displayプロパティ
   - block
   - inline
   - inline-block
   - none

5. **05_flexbox.css** - Flexbox
   - フレックスコンテナとフレックスアイテム
   - justify-content
   - align-items

6. **06_grid.css** - CSS Grid
   - グリッドコンテナの定義
   - grid-template-columns/rows
   - gap

7. **07_typography.css** - タイポグラフィ
   - フォントファミリー、サイズ、ウェイト
   - テキスト配置、行の高さ
   - テキスト装飾

8. **08_position.css** - positionプロパティ
   - static、relative、absolute
   - fixed、sticky
   - z-index

9. **09_transition.css** - CSSトランジション
   - プロパティの変化をアニメーション
   - イージング関数
   - ホバーエフェクト

## 使い方

各CSSファイルには、スタイルの定義とコメントで説明が記載されています。
HTMLファイルと組み合わせて使用することで、スタイルの適用を確認できます。

例:
```html
<link rel="stylesheet" href="css/01_selector.css">
```
