# HTMLサンプル集

このフォルダには、HTMLの基本的な要素を学ぶためのサンプルファイルが含まれています。
各ファイルは1つの概念に焦点を当てています。

## サンプル一覧

1. **01_basic_structure.html** - HTMLの基本構造
   - HTMLドキュメントの最小構成
   - head要素とbody要素の役割

2. **02_headings.html** - 見出しタグ
   - h1からh6までの見出しレベル

3. **03_paragraph.html** - 段落タグ
   - p要素を使った段落の作成

4. **04_link.html** - リンクタグ
   - a要素を使ったハイパーリンクの作成
   - 新しいタブで開く方法

5. **05_image.html** - 画像タグ
   - img要素を使った画像の表示
   - alt属性の重要性

6. **06_list.html** - リストタグ
   - ul要素（順序なしリスト）
   - ol要素（順序ありリスト）

7. **07_form.html** - フォームタグ
   - form要素を使ったユーザー入力の収集
   - input要素とlabel要素

## 使い方

各HTMLファイルをブラウザで開くと、その要素の動作を確認できます。
ファイル内のコメントで、各要素の使い方が説明されています。
